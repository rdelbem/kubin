Kubin Galleryv1

A vanilla JavaScript image gallery with lightbox

This is a demo and a basic set up structure

Kubin Gallery takes pride in giving you total control over its features

Kubin Gallery requires two files in order to work properly, a js and a css.

Currently, we are offering the main file, the js one, from our GitHub deploy page, from the link: https://rdelbem.github.io/kubin/kubin-gallery-v1.js

Our css file link is: https://rdelbem.github.io/kubin/kubin-styles-v1.css

Our repo is https://github.com/rdelbem/kubin

Follow the basic markdow shown bellow in order to embed Kubin to your HTML template.

<div class="kubin-container">
    <div class="kubin-principal"></div>
    <div class="kubin-thumbs">
     <ul class="kubin-img-gallery">
      <li>
       <img class="kubin-gallery-thumb" src="here goes your url" />
      </li>
     </ul>
    </div>
   </div>

Insert the following script on your html in order to tweak features.
It must come before the kubin-gallery-v1.js embed. 

<script type="application/javascript">
   var setOptions = {
    overlay: "#000000",
    overlayAlpha: "99",
    overlayBlur: true,
    baseURL: "https://rdelbem.github.io/kubin/",
    fullSizePath: "images/fullsize/",
    lightBox: true,
    zoom: true,
    bigImgsPostFix: false,
   };
   </script>

Configs

  • overlay controls the color of the lightbox background
  • overlayAlpha controls the transparency of the lightbox background
  • overlayBlur controls if there is blur on the lightbox background. It must be true or false.
  • baseURL provides the main address where are the images. It must end with a /
  • fullSizePath provides the folders where are the images. It must not start with a / and must end with an /
  • lightbox controls if there will be lightbox or not. It must be a true or false value.
  • zoom turn on or off the zoom on mouve hover. It must be a true or false value
  • bigImgsPostFix use this if your big images have a post fix. Natively Kubin Gallery will search for big images with the same name as the thumbs.
